# 🎨 ArtisCard - Система управления портфолио

**ArtisCard** - это веб-приложение для создания и управления профессиональными портфолио с системой модерации. Платформа позволяет пользователям создавать свои портфолио, а администраторам - проверять и публиковать их.

## 🚀 Возможности

### 👤 **Для пользователей**
- ✅ **Регистрация и авторизация** в системе
- 📝 **Создание и редактирование** профессионального портфолио
- 👁️ **Просмотр** своего опубликованного портфолио
- 📄 **Генерация резюме** в формате HTML
- 📊 **Отслеживание статуса** портфолио (на проверке/опубликовано)
- 🔒 **Личный кабинет** с навигацией

### 👑 **Для администратора**
- 📋 **Панель управления** всеми пользователями
- ✏️ **Редактирование и публикация** портфолио пользователей
- 📊 **Статистика** по статусам портфолио
- 🔍 **Просмотр списка** пользователей с фильтрацией по статусу
- ✅ **Одобрение/отклонение** портфолио

## 🛠️ **Технологический стек**

**Backend:**
- **Python 3.9+**
- **Flask** (веб-фреймворк)
- **SQLite3** (база данных)
- **Jinja2** (шаблонизатор)

**Frontend:**
- **HTML5 + CSS3**
- **JavaScript** (нативный)
- **Bootstrap 5** (стилизация)
- **Font Awesome 6** (иконки)
