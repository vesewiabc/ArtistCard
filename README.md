# 🎨 ArtisCard - Система управления портфолио

**ArtisCard** - это веб-приложение для создания и управления профессиональными портфолио с системой модерации. Платформа позволяет пользователям создавать свои портфолио, а администраторам - проверять и публиковать их.

---

## 🚀 Основные возможности

### 👤 Для пользователей
- ✅ **Регистрация и авторизация** в системе
- 📝 **Создание и редактирование** профессионального портфолио
- 👁️ **Просмотр** своего опубликованного портфолио
- 📄 **Генерация резюме** в формате HTML
- 📊 **Отслеживание статуса** портфолио (на проверке/опубликовано)
- 🔒 **Личный кабинет** с интуитивной навигацией

### 👑 Для администратора
- 📋 **Панель управления** всеми пользователями
- ✏️ **Редактирование и публикация** портфолио пользователей
- 📊 **Статистика** по статусам портфолио
- 🔍 **Просмотр списка** пользователей с фильтрацией по статусу
- ✅ **Одобрение/отклонение** портфолио

---

## 🛠️ Технологический стек

### 🐍 **Backend**
- **Python 3.9+** - основной язык программирования
- **Flask 2.3.2** - легковесный веб-фреймворк
- **SQLite3** - встроенная база данных

### 🎨 **Frontend**
- **HTML5** - семантическая разметка
- **CSS3** - каскадные таблицы стилей
- **JavaScript (ES6+)** - интерактивность на стороне клиента


### 🗄️ **База данных**
- **SQLite3** - легковесная реляционная БД
- **Схема данных:**
  - Таблица `users` (пользователи)
  - Таблица `user_profiles` (профили портфолио)
- **Миграции:** Автоматическое создание таблиц при запуске

### 🔧 **Инструменты разработки**
- **Git** - система контроля версий
- **GitHub** - хостинг репозитория
- **VS Code** - среды разработки
- **SQLite Browser** - просмотр базы данных

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/ваш-username/ArtisCard.git
cd ArtisCard
```

2. Установка зависимостей

```bash
pip install flask
```

3. Запуск приложения

```bash
python app.py
```

4. Открытие в браузере

Перейдите по адресу: http://localhost:5555
